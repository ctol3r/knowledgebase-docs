# Contributing to Truvera Docs

We welcome pull requests that improve the documentation or fix problems. To propose a change:

1. Fork the repository and create your branch from `main`.
2. Make your changes with clear commit messages.
3. Ensure that your contribution follows our [Code of Conduct](open-source-community/code_of_conduct.md).
4. Open a pull request describing the proposed improvement.

For questions or larger contributions, please open an issue first to discuss the changes.
